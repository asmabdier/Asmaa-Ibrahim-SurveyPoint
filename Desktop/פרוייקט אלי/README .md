# 📐 SurveyPoint — המרת תיק חישובים לקובץ קואורדינטות ולציור AutoCAD

> כלי עבודה חכם למודדים — מהתיק לשטח בלחיצת כפתור

---

## 🧭 רקע ומטרה

מודדים מקרקעין עובדים מדי יום עם **תיקי חישובים** המכילים קואורדינטות של נקודות מדידה.  
תהליך ההמרה הידני של נתונים אלו לקובץ CSV מסודר ולאחר מכן לציור ב-AutoCAD הוא:

- ממושך ומייגע
- מועד לטעויות אנוש בהעתקה
- בזבזן בזמן בכרב תחומים — ובפרט בעבודות **שחזור גבולות**

**SurveyPoint** נוצרה כדי לפתור בדיוק את הבעיה הזו.  
האפליקציה מקבלת כקלט תיק חישובים, ומייצרת באופן אוטומטי שני פלטים מוכנים לעבודה:

1. **קובץ CSV** — טבלת קואורדינטות מסודרת לפי שם נקודה, Y, X
2. **קובץ DWG** — ציור AutoCAD עם כל הנקודות מסומנות ומתויגות

## 📥 קלט — תיק חישובים

האפליקציה מקבלת קבצי תיק חישובים מהתוכנות הנפוצות:

- `.dat` / `.txt` — פורמט טקסט מובנה
- `.csv` — קובץ גיליון מופרד פסיקים
- `.job` — קבצי מדידה ייחודיים

---

## 📤 פלטים

### 1. קובץ CSV (`.csv`)

סדר העמודות בקובץ הפלט:

```
שם נקודה, Y (צפון), X (מזרח)
```

לדוגמה:
```csv
שם נקודה,Y,X
101,665123.780,183452.340
102,665130.200,183460.120
103,665118.560,183455.890
```

- מופרד בפסיקים
- כל נקודה בשורה נפרדת
- ניתן לייבא ישירות ל-AutoCAD, Excel וכלי GIS

### 2. קובץ DWG (AutoCAD)

- כל נקודה מוצגת כ-**Point entity**
- לצד כל נקודה — **תווית טקסט** עם המזהה שלה
- נקודות מסודרות ב**שכבות (Layers)** לפי סוג
- ניתן לפתוח ישירות ב-AutoCAD / BricsCAD / ZWCAD

## 🏗️ שימושים מקצועיים

האפליקציה מתאימה במיוחד לתהליכים הבאים:

- **שחזור גבולות** — ייבוא נקודות גבול היסטוריות לציור מהיר
- **תוכניות מדידה** — הכנת בסיס לתוכניות רישום ואיחוד חלקות
- **סיור שטח** — המרה מהירה של נתוני שדה לציור עבודה
- **בדיקת תוויות** — השוואה בין נקודות ממדידות שונות
- **ארכיון דיגיטלי** — שמירת תיקי חישובים בפורמט אחיד ומובנה
