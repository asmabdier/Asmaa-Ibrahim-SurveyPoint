# 📐 SurveyPoint — המרת תיק חישובים לקובץ קואורדינטות ולציור AutoCAD

> אפליקציה שמקבלת תיק חישובים של מודד ומייצרת אוטומטית קובץ CSV וקובץ DWG מוכן לעבודה ב-AutoCAD.

---

## 🔴 הבעיה שפותרים

מודדים מקרקעין עובדים מדי יום עם **תיקי חישובים** המכילים קואורדינטות של נקודות מדידה.  
תהליך ההמרה הידני של נתונים אלו לקובץ CSV ולאחר מכן לציור ב-AutoCAD הוא:

- **ממושך** — לוקח עשרות דקות לכל תיק
- **מועד לשגיאות** — העתקה ידנית של קואורדינטות גורמת לטעויות קריטיות בשטח
- **בלתי יעיל** — במיוחד בעבודות שחזור גבולות שדורשות עשרות תיקים בו-זמנית

**SurveyPoint** נוצרה כדי לפתור בדיוק את הבעיה הזו — מהתיק לשטח בלחיצת כפתור.

---

## 👤 קהל יעד

**מודד מקרקעין מוסמך** — אדם שעובד בחברת מדידה ומבצע מדידות שטח, שחזורי גבולות ותוכניות רישום.  
הוא עובד יומיומית עם AutoCAD ועם תיקי חישובים דיגיטליים, ומבזבז זמן רב על המרות ידניות שחוזרות על עצמן.

---

## 🗄️ מקור הנתונים

**נתוני קואורדינטות פתוחים — מפ"י (מרכז המיפוי של ישראל):**

🔗 https://data.gov.il/dataset/geoproduct

הנתונים כוללים קואורדינטות ITM (Israeli Transverse Mercator) של נקודות מדידה רשמיות בפורמט CSV.  
כל רשומה מכילה: מזהה נקודה, Y (צפון), X (מזרח).

---

## 🏗️ ארכיטקטורה טכנית

**סוג המודל:** עיבוד נתונים (Data Pipeline) + זיהוי ערכים חריגים (Anomaly Detection — Isolation Forest)

| רכיב | פירוט |
|---|---|
| **Input** | קובץ תיק חישובים (.csv / .dat / .txt) |
| **עיבוד** | פרסור, ניקוי, זיהוי ערכים חריגים |
| **Output 1** | קובץ CSV מסודר: שם נקודה, Y, X |
| **Output 2** | קובץ DWG עם נקודות מסומנות ב-AutoCAD |
| **ספריות** | `pandas`, `streamlit`, `plotly`, `scikit-learn`, `ezdxf` |

## 📖 User Stories

1. **כמודד מקרקעין**, אני רוצה **להעלות תיק חישובים ולקבל מיידית קובץ CSV מוכן**, כדי **לחסוך 30 דקות עבודה ידנית לכל תיק**.

2. **כמודד מקרקעין**, אני רוצה **לקבל קובץ DWG עם הנקודות מסומנות**, כדי **להמשיך ישירות לעבודה ב-AutoCAD בלי ייבוא ידני**.

3. **כמודד מקרקעין**, אני רוצה **שהמערכת תסמן נקודות חשודות (ערכים חריגים)**, כדי **לאתר טעויות הקלדה לפני שהן גורמות לשגיאות בשטח**.

## 📥 קלט — תיק חישובים

האפליקציה מקבלת קבצי תיק חישובים מהתוכנות הנפוצות:

- `.dat` / `.txt` — פורמט טקסט מובנה
- `.csv` — קובץ גיליון מופרד פסיקים
- `.job` — קבצי מדידה ייחודיים

---

## 📤 פלטים

### 1. קובץ CSV (`.csv`)

סדר העמודות בקובץ הפלט:

```
שם נקודה, Y (צפון), X (מזרח)
```

לדוגמה:
```csv
שם נקודה,Y,X
101,665123.780,183452.340
102,665130.200,183460.120
103,665118.560,183455.890
```

### 2. קובץ DWG (AutoCAD)

- כל נקודה מוצגת כ-**Point entity**
- לצד כל נקודה — **תווית טקסט** עם המזהה שלה
- נקודות מסודרות ב**שכבות (Layers)** לפי סוג
- ניתן לפתוח ישירות ב-AutoCAD / BricsCAD / ZWCAD

## 🤖 מודל ML — Isolation Forest

**סוג המודל:** Anomaly Detection (זיהוי ערכים חריגים)

| פרמטר | ערך |
|---|---|
| אלגוריתם | Isolation Forest |
| contamination | 0.05 (5% חריגים צפויים) |
| random_state | 42 |
| train/test split | 80%/20% |

**מה המודל עושה:**
- מאומן על קואורדינטות Y, X של כל תיק חישובים
- מזהה נקודות שנראות חריגות ביחס לשאר
- מסמן נקודות חשודות ⚠️ שעלולות להיות טעויות הקלדה

**תוצאות אימון:**
- תיקייה 1: 192 train, 49 test, 2 חריגים
- תיקייה 2: 67 train, 17 test, 4 חריגים
- תיקייה 3: 27 train, 7 test, 0 חריגים
- תיקייה 4: 132 train, 33 test, 0 חריגים
